// Instructions for running a MongoDB database in Docker

// 1. Start Docker Desktop.

// 2. In Docker Desktop, open the terminal in the bottom-right corner
//    (Docker CLI) and run the following command to create
//    and start a container with the database:
//
//    docker run -d --name pokemonDB -p 27017:27017 mongo:6.0
//
//    - container name: pokemonDB
//
// 3. The Node.js application connects to the database at:
//
//    mongodb://127.0.0.1:27017/pokemon_wiki
//
// 4. Stopping and starting the container again (optional):
//
//    docker stop pokemonDB
//    docker start pokemonDB
